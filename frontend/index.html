DeepSeek Market Analyzer

爬蟲模式 · 無 API 費用

市場資料 (JSON)

使用說明

1. 點「自動抓取市場資料」拉取 BTC/ETH 即時價格 + 恐懼貪婪指數
2. 或自己在左側 JSON 編輯器貼入任何市場資料
3. 點「送出分析」— 後端用 Selenium 把 JSON 注入 DeepSeek 聊天框
4. 等待 DeepSeek 回應（約 10-30 秒），結果顯示在下方

設定 Chrome Profile
後端需要已登入 DeepSeek 的 Chrome Profile。
在終端設定環境變數：
CHROME_PROFILE=/path/to/chrome/profile python app.py

或設 HEADLESS=false 讓瀏覽器視窗可見以手動登入。
